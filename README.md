# PEML-reproducibility
Reproducibility materials for PEML

## Contents
- Dataset construction code for CIFAR-100, ImageNet-100, and CIFAR100-LT
- ImageNet-100 class list / subset-definition files
- Synthetic-data generation code and configuration

## Data sources
This repository does not redistribute third-party raw image datasets.
Users should obtain CIFAR-10/100 and ImageNet from their official sources and then use the scripts in this repository to reproduce the experimental setup.

## Main file
- `data_build.py`: cleaned dataset-construction code for release
